# Wajir Water Point Management Information System

A web-based system for registering and tracking water points in Wajir County, northeastern Kenya.

## Built With

PHP, MySQL, HTML, CSS, XAMPP

## What It Does

Allows authorized users to register water points, track their operational status, log breakdowns and repairs, and generate reports. Supports two roles: Admin and Operator.

## How to Run Locally

1. Install XAMPP and start Apache and MySQL
2. Place the project folder inside C:/xampp/htdocs/
3. Create a database called water_point_db in phpMyAdmin
4. Import the SQL file to set up the tables
5. Visit http://localhost/waterpoint/login.php in your browser
